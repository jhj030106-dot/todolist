📝 Todo List 프로젝트

📖 프로젝트 소개
순수 자바스크립트(Vanilla JS)를 활용하여 구현한 심플하고 **할 일 관리(Todo List)**입니다. 


✨ 구현한 주요 기능
* **할 일 추가**: 텍스트 입력 후 `[+]` 버튼 또는 `Enter` 키를 눌러 새로운 할 일을 추가할 수 있습니다. (빈 칸 입력 방지 기능 포함)
* **할 일 완료**: `[완료]` 버튼을 통해 할 일의 상태를 완료(취소선)로 변경할 수 있습니다.
* **할 일 수정**: `[수정]` 버튼을 눌러 내용을 변경하고 `[저장]`할 수 있습니다. (수정 중 다른 동작 수행 시 자동 저장되는 방어막 로직 적용)
* **할 일 삭제**: `[삭제]` 버튼 클릭 시, `confirm` 확인창을 띄워 실수로 인한 데이터 영구 삭제를 방지하는 안전장치를 마련했습니다.
* **전체 초기화**: `[초기화]` 버튼으로 모든 할 일 목록과 저장소 데이터를 한 번에 비울 수 있습니다.
* **데이터 영구 저장**: 브라우저를 종료하거나 새로고침해도 작성한 할 일 목록이 그대로 유지됩니다.

🛠 적용된 자바스크립트 핵심 기술
* **DOM Manipulation**: `querySelector`, `innerHTML` 등을 사용하여 HTML 요소를 동적으로 생성하고 화면에 렌더링합니다.
* **LocalStorage**: 브라우저의 웹 스토리지 API를 활용하여 사용자 데이터를 브라우저 내부에 영구적으로 보존합니다. (`setItem`, `getItem`, `removeItem`)
* **JSON Object**: 데이터를 저장하고 불러올 때 `JSON.stringify()`와 `JSON.parse()`를 사용하여 자바스크립트 배열 객체와 문자열 간의 변환을 안전하게 수행합니다.
* **Event Handling**: `onclick`, `onkeydown` 이벤트를 통해 사용자의 동작(버튼 클릭, 엔터키 입력 등)을 감지하고 적절한 함수를 호출합니다.
* **Array Methods**: `forEach`, `push` 등의 배열 메서드를 적극적으로 활용하여 다수의 할 일 목록 데이터를 효율적으로 순회하고 조작합니다.
