📝 Todo List 프로젝트

## 📖 프로젝트 소개
매일의 일상을 체계적으로 기록하고 관리할 수 있도록 돕는 할 일 보관함 프로그램입니다. 사용자가 마우스를 잘못 조작하여 소중한 기록을 한순간에 날려버리지 않도록, 실수를 방지하는 안전장치를 겹겹이 보완하여 안심하고 쓸 수 있게 만들었습니다.

## ✨ 주요 기능
* ✅ **기본 기능 완벽 구현**: 할 일 추가, 완료(취소선), 수정, 삭제
* 🛡️ **데이터 보호 (안전장치)**: 
  * 삭제 전 경고창 띄우기 (`confirm`)
  * 항목 수정 중 다른 버튼 클릭 시 텍스트 자동 저장
* 💾 **데이터 영구 보존**: 브라우저를 껐다 켜도 데이터 유지
* ⌨️ **편의성**: `Enter` 키로 즉시 추가, 빈 칸 입력 방지, 전체 초기화 기능

## 🛠 핵심 자바스크립트 기술
* **DOM 조작**: `querySelector`, `innerHTML`로 화면을 동적으로 그리기
* **데이터 저장소**: 브라우저 `LocalStorage` 및 `JSON` 데이터 변환
* **이벤트 처리**: `onclick`, `onkeydown`을 통한 클릭 및 키보드 감지
* **배열 활용**: `forEach`, `push`, `filter` 등 자바스크립트 배열 메서드 사용
